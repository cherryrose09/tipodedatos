Tipos de Datos

Los tipos de datos JavaScript se dividen en dos grupos: tipos primitivos y tipos objeto.

Los tipos primitivos incluyen:cadenas de texto (String), variables booleanas cuyo valor puede ser true o false (Boolean) y números (Number). Además hay dos tipos primitivos especiales que son Null y Undefined. Los tipos objeto son datos interrelacionados, no ordenados, donde existe un nombre de objeto y un conjunto de propiedades que tienen un valor.

TIPOS PRIMITIVOS

  • String:

    Cadenas de texto
    "tarta de plátano vegana con masa casera";

  • Number:

    Valores numéricos
    var peso = 54;
    var pesoExacto = 54.3;

  • Boolean:

    true ó false
    var x = false;
    if (x) { // este código no se ejecuta };

  • Null:

    El tipo Null tiene el valor: null.

  • Undefined:

    Una variable a la cual no se le haya asignado valor tiene el valor undefined.
    var saludo; console.log(typeof saludo); // "undefined"

TIPOS OBJETO

  • Tipos predefinidos de JavaScript:

    Date (fechas)
    RegExp (expresiones regulares)
    Error (datos de error)

  • Arrays:

    Serie de elementos o formación tipo vector o matriz. Lo consideraremos un objeto especial.

  • Objetos especiales:

    Objeto global
    Objeto prototipo
    Otros
